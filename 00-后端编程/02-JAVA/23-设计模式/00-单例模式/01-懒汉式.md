

```java
public class Test {

  // 1. 私有化的构造器
  private Test() {
  }

  // 2. 声明当前类对象, 没有初始化
  // 3. 要求此对象也必须声明为静态的
  private static Test instance = null;

  // 3. 提供公共的静态的方法, 返回类对象
  public static Test getInstance() {
    // 这样解决线程安全问题, 效率高
    synchronized(Test.class) {
      if (instance == null) {
      	instance = new Test();
    	}
    }
    return instance;
  }
}
```
