合并两个或多个 SELECT 语句的结果集



## `union all`

`union all`是把结果直接合并在一起, 不去重

```sql
(select id from news)
union all
(select id from you);//推荐将select语句用（）包起来
```

## `union`

union是将union all的后的结果进行一次distinct，去重复记录后的结果

语法

```sql
(select 语句1)
union
(select 语句2)

```
