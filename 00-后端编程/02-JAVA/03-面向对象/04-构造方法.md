构造方法的名称必须和所在的类名称完全一样, 就连大小写也要一样

构造方法不要写返回值类型, 连`void`都不能写

构造方法不能`return`

如果没有写任何构造方法, 会默认加一个没有参数的构造方法,  方法体什么事情都不做.

一旦写了至少一个构造方法, 那么将不会再默认添加构造方法.

```java
public class Demo1 {
    public String name;
    public int age;

    public Demo1() {
    }
    public Demo1(String name, int age) {
        this.name = name;
        this.age = age;
    }
}


```
