一个Go主线程上, 可以起多个协程, 可以这么理解, 协程是轻量级的线程(编译器做优化)

Go协程的特点:

1. 有独立的栈空间
2. 共享程序堆空间
3. 调度由用户控制
4. 协程是轻量级别的线程



```go
func test() {
	for i:=0;i<10 ;i++  {
		fmt.Println("test() hello")
		time.Sleep(time.Second)
	}
}

func main() {
	go test() // 开启一个协程

	for i:=0;i<10 ;i++  {
		fmt.Println("main() hello")
		time.Sleep(time.Second)
	}
}

```
