## 值传递

不管是指针, 引用类型, 还是其他类型的形参, 都是`值拷贝`.

区别无非是拷贝目标对象, 还是拷贝指针.在函数调用前, 会维形参和返回值分配内存空间, 并将实参拷贝到形参内存



## 变参

```go

func test(a ...int)  {
	fmt.Println(a) // [1 2 3 4]
	fmt.Printf("%T", a) // []int
}

func main() {
	test(1,2,3,4)
}
```

变参本质上就是一个`切片`.只能接收一到多个同类型参数, 且必须放在列表尾部.

将切片作为参数时, 必须进行展开操作. 如果是数组, 先转成切片.

```go
func main() {
	arr := [4]int{1,2,3,4}
	test(arr[:]...) // 转换为 slice 后展开
}
```
