## 函数返回多个值

![](https://ws2.sinaimg.cn/large/006tKfTcly1g0y18myp7aj30zg08sdgx.jpg)



```go
package main

func div(a, b int) (q, r int) {
	return a/b, a%b
}

func main() {
	q, _ := div(4, 5)  // _ 来处理不想要的返回值
	println(q)
}
```

## 可变参数列表

```go

func sum(numbers ...int) int {
	sum:=0
	for i := range numbers {
		sum +=numbers[i]
	}
	return sum
}

```
