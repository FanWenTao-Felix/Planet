## 特点

当第一次用到本类时, 静态代码块执行唯一的一次.

静态内容总是优先于非静态, 所以静态代码块比构造方法先执行.

## 典型用途

用来一次性的对`静态成员变量`进行赋值



代码示例

```java
public class Demo2 {
    // 静态代码块
    static {
        System.out.println("执行-静态代码块");
    }

    public Demo2() {
        System.out.println("执行-构造方法");
    }
}


// 调用2次
new Demo2();
new Demo2();
// 输出
执行-静态代码块
执行-构造方法
执行-构造方法


```
