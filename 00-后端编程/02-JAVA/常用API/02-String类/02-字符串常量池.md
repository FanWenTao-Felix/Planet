程序中直接写上的`双引号字符串`,    就是再字符串常量池中.

## `==` 问题

对基本类型来说,  `==`是进行数值的比较

对引用类型来说, `==`是进行`地址值`的比较

```java
String str1 = "abc";
String str2 = "abc";
System.out.println(str1 == str2); // true, str1 和 str2 共享

char[] charArr = {'a', 'b', 'c'};
String str3 = new String(charArr);
System.out.println(str1 == str3); // false str3不共享
```
