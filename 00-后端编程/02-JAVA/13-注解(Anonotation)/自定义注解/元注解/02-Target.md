`Target` 是一个数组

## `@Target`: 描述注解能够作用的位置

说明了Annotation所修饰的对象范围：



`ElementType`取值:

```
FIELD 被用于描述域

PACKAGE 被用于 packages

TYPE 被用于 types（类、接口、枚举、Annotation类型）

METHOD/CONSTRUCTOR 被用于 类型成员（方法、构造方法、成员变量、枚举值）

PARAMETER 被用于 方法参数和本地变量（如循环变量、catch参数）
```
