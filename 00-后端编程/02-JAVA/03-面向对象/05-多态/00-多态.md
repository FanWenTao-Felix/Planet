继承是子类使用父类的方法，而多态则是父类使用子类的方法。一般，我们使用多态是为了避免在父类里大量重载引起代码臃肿且难于维护。

多态: 父类引用指向子类的对象

```
父类名称 对象名 = new 子类名称();
或者:
接口名称 对象名 =  new 实现类名称();
```

```java
public class Fu {
  public int num = 100;
    public void eat() {
        System.out.println("父类 eat");
    }
}
```

```java
public class Zi extends Fu {
  public int num = 200;
    @Override
    public void eat() {
        System.out.println("子类eat");
    }
}
```

```java
public static void main(String[] args) {
  Fu fu = new Zi(); // 多态的写法
}
```
