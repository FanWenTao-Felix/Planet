## 自增||自减

### 说明1

只能当做一个独立的语言使用,**不能如下使用**

```go
b := a++
b := a--
```

正确写法:

```go
a := 1
a++
b := a
fmt.Println(b)
```

### 说明2

**只有**`a++`和`a—`,  

**没有**`++a`和`—a`

## 说明3

不能作为表达式使用

```go
x:=100
if (x++) > 100 { //unexpected ++, expecting
  println(11)
}
```
