## Map集合特点

Map集合是一个双列集合, 一个元素包含两个值(一个key, 一个 value)

Map集合中的元素, key和value的数据类型可以相同, 也可以不同

Map集合中的元素, key是不允许重复的, value是可以重复的.

Map集合中的元素, key和value是一一对应
