常量:  在程序运行期间, 固定不变的量.

常量的分类:

## 字符串常量

凡是用双引号引起来的部分, 例如"ABC"

## 整数常量

直接写上的数字, 没有小数点, 例如 100, 200 , 0, -100

## 浮点数常量

直接写上的数字, 有小数点, 例如 2.5, 0.0

## 字符常量

凡是用单引号引起来的单个字符,  例如 `'A', 'b'`

## 布尔常量

true, false

## 空常量

null,  代表没有任何数据.

```java
// System.out.println(null); // 空常量不能直接用来打印输出
```
