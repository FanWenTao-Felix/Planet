## java.util.ConcurrentModificationExcep

并发不安全问题

```java
public static void main(String[] args) {
  List<String> stringList = new ArrayList<>();
  for (int i = 0; i < 30; i++) {
    new Thread(()->{
      stringList.add(UUID.randomUUID().toString().substring(0,8));
      System.out.println(stringList);
    },"线程"+i).start();
  }
}
```
