使用步骤:

1. 创建`Cookie`对象, 绑定数据

   ```
   new Cookie(String name, String value)
   ```

2. 发送Cookie

   `reponse.addCookie(Cookie cookie)`

3. 获取Cookie

   `Cookie[] request.getCookies()`
