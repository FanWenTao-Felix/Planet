区别:

`#{}`: 是以预编译的形式,将参数设置到sql语句中, 防止sql注入

`${}`: 取出的值 直接拼装在sql语句中(sql拼接), 会有安全问题

```sql
select * from ${year}_table where xxx; // 需要用 ${}
select from tmp_table order by ${f_name} desc; // 需要用 ${}
select * from test_table where name like '%${name}%'; // 模糊查询
```
