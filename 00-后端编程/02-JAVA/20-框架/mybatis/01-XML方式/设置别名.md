

## typeAlias

劣势: 这种只能给某一个设置别名

```xml
<typeAliases>
	<typeAlias alias="demo_02_user" type="com.zongqi.demo_02.bean.User"/>
</typeAliases>

```



## package

给当前包下的`所有类` 批量起别名,  默认`类名`就是`别名`

```xml
<typeAliases> 
  <package name = "com.mybatis.Bean" />
</typeAliases>
```

如果类名有冲突, 可以使用`@Alias("name")`, 重写设置别名.

```java
@Alias("hello")
public class DemoBean() {
  
}
```
