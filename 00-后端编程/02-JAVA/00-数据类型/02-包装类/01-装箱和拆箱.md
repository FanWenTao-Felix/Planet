基本类型与对应的包装类对象之间, 来回转换的过程称为`装箱`和`拆箱`.

## 装箱

基本类型 ----> 对应的包装类对象

## 拆箱

包装类对象 ----> 对应的基本类型

## 自动装箱和自动拆箱

```java
// 自动装箱,相当于 Integer i = Integer.valueOf(4)
Integer i = 4;
// 等号右边: 将i对象转换成基本数值(自动拆箱), i.intValue() +5
i = i + 5;

// 加法运算完成后, 再次装箱, 把基本数值转成对象
```
