定义格式

```
修饰符 <泛型变量>  返回值类型 方法名(参数) { }
```

```java
public class Demo4 {

    public <MVP> void method01(MVP m) {
        System.out.println(m);
    }

    // 泛型静态方法
    public static <S> void method2(S s) {
        System.out.println(s);
    }
}
```

```java
Demo4 demo4 = new Demo4();
// 传递是什么类型, 泛型就是什么类型
demo4.method01("hello");
Demo4.method2(1000);
```
