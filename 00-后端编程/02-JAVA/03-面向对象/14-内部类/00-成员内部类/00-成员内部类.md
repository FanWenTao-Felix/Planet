## 成员内部类

- `static`成员内部类
- 非`static`成员内部类

## 格式

```java
public class Test {
  	// 成员内部类
  	private class Inner {
      //....
    }
 
}
```

## 访问规则

内用外, 随意访问

外用内, 需要实例化内部类对象
